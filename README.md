# Education-Level-by-State
This repository contains an analysis of education levels across U.S. states and territories for 2020. The project includes visualization, percentage calculations, ranking, and composite scoring of educational attainment.

# Dataset

Dataset contains education data by state (cd) for 2020.

# Columns:

- Bachelors_degree_or_higher – Number of people with a bachelor’s degree or higher

- high_school_or_some_degree – Number of people with high school or some college

- Less_than_high_school_graduate – Number of people with less than high school education

Dataset From Kaggle

# Key Insights

- Bachelor’s degree attainment varies widely by state; some states like IL, NY, and CA lead, while others lag behind.

- Weighted Education Score captures both population size and education level for holistic ranking.

- Visualization tools like stacked bars and heatmaps help quickly identify patterns and outliers.

- Data can be used for policy analysis, workforce planning, and educational investment
